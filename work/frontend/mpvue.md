# 微信小程序框架和 mpvue 开发

## mpvue 脚手架可以依赖 vue-cli 搭建
```bash
  $ npm install -g vue-cli
  $ vue init mpvue/mpvue-quickstart my-project
  $ cd my-project
  $ npm install
  $ npm run dev
  # 之后启动微信开发者工具，引入项目
```
